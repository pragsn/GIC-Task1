     *🤖AI HEALTH PREDICTION SYSTEM*

* In this Project I have used python as backend and React as Frontend.
* This a simple website with easy navigation to get remarks from _AI_ using the blood test values and email address gievn in the user interface.
* I have also used 🚀 **_fastAPI_** for _**CRUD**_ operations as python is used in backend.
* uvicorn to run the virtual environment.
* for persistent Database i have used 📀 _**SQLite**_.
* The backend server status can be checked using ∷ http://127.0.0.1:8000
* The patient details can be checked from backend using ∷ http://127.0.0.1:8000/patients or ∷ http://127.0.0.1:8000/patients
* The patient details are stored in 💼 _**patients.db**_ file.


WORKFLOW:
1. start the frontend, backend applications and enter the details in frontend
2. ✉️ _**Email address**_ cannot be invalid
3. 🗓️ _**Date of Birth**_ cannot be a future value
4. 🩸 The blood test values(Haemoglobin, Cholestrol, Glucose) cannot exceed their respective ranges.
5. save the details and wait for 🤖 _**Google-Gemini-AI-API**_ to respond and give remarks.
6. use the edit and delete options to change the data.
7. Hence the Create, Read, Update and Delete operations are satisfied.
